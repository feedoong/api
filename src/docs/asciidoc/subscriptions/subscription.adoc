=== 구독 목록 조회
이 API 엔드포인트는 사용자의 구독 목록을 조회합니다.

*GET* - /v2/subscriptions

==== 요청
- 요청 파라미터는 `page`, `size`, `sort`, `direction` 을 포함합니다.
- 인증을 위해 `Authorization` 헤더가 필요합니다.
